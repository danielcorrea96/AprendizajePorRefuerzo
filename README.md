# AprendizajePorRefuerzo

## Instalar librerías necesarias

Se instala dependiendo del entorno si es **COLAB** se instala todo lo necesario para el taller (creacion, compilacion, entrenamiento y pruebas del agente), si es local solo instala lo necesario para visualizar las graficas.

## Importar Librerías

Es necesario agregar un parche de compatibilidad k2 para temsorflow 2.18 + Keras-rl2, con esto se evitan fallos al importar las librerías.
